# E-commerce Product Scraper 🛍️

This project contains a sample dataset of e-commerce products, formatted as if scraped from a real website.

### 📦 Collected Fields:
- Product Title  
- SKU  
- MPN  
- Price  
- Image URL  
- Description

---

## 🛠️ Tools Used
- Python 🐍
- pandas
- Excel for output formatting

---

## 📁 Output
Saved in: `final_ecommerce_products.xlsx`
